# To-Do List

A sleek and modern to-do list application built with Electron.

<img width="450" height="556" alt="Screenshot 2025-12-31 191702" src="https://github.com/user-attachments/assets/7ba8d64d-1f9a-45ae-91a1-0312f9889357" />


## Features

- Create and manage tasks with a beautiful, neon-styled interface
- Mark tasks as complete with an animated checkbox
- Local data storage using SQLite
- Cross-platform desktop application (Windows, macOS, Linux)

## Technology Stack

- Electron
- HTML/CSS/JavaScript
- SQLite for data persistence

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/hyper0939/To-Do-List.git
   cd To-Do-List
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the application:
   ```
   npm start
   ```

## Building the Application

To create a distributable package:

```
npm run make
```

This will generate distributable packages in the `out/make` directory.
